# 🚔 Urban CrimeSense

**Urban CrimeSense** is a spatio-temporal crime hotspot detection and prediction framework developed to analyze urban crime patterns using machine learning and geospatial techniques. The project leverages historical crime data to identify high-risk areas, uncover spatial and temporal trends, and support data-driven decision-making for crime prevention.

---

## 📌 Project Overview

The objective of this project is to predict crime hotspots by integrating spatial and temporal features extracted from historical crime records. The framework combines data preprocessing, feature engineering, machine learning models, and geospatial visualization to provide actionable insights into urban crime patterns.

### Key Features

* Data cleaning and preprocessing
* Spatial and temporal feature engineering
* Crime hotspot prediction using Machine Learning
* Model evaluation and comparison
* Geospatial visualization of crime hotspots
* Research-oriented implementation
